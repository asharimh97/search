# STKI Search

Implementasi pembuatan search engine berdasarkan seluruh materi yang didapatkan dari mata kuliah Sistem Temu Kembali Informasi dengan menggunakan Python.

## Petunjuk instalasi
```
pip install -r requirements.txt
```

## Running 
```
python run.py
```
